# 🛒 Django Ecommerce Website

This is a simple Ecommerce web application built using Django.

## Features
- Product listing page
- Product detail page
- Add to cart
- Increase / decrease cart quantity
- Remove items from cart
- Checkout page

## Tech Stack
- Python
- Django
- HTML
- CSS
- SQLite

## How to Run

1. Clone the repository
git clone https://github.com/Sailaja505/Ecommerce.git

2. Navigate to project folder
cd Ecommerce

3. Run the server
python manage.py runserver

Then open:
http://127.0.0.1:8000/
